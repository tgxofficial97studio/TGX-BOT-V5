# TGX Bot V5.1 Help GUI

Tambahan ini membuat `/help` menjadi GUI Discord interaktif seperti panel premium.

Setelah deploy ulang di Railway, jalankan:

```bash
npm run deploy:guild
```

Lalu coba `/help` di Discord.

Catatan: tombol Discord tidak bisa mengetik slash command otomatis di kolom user. Tombol akan menampilkan command dan cara pakainya.
